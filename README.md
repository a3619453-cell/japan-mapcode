# 日本 Mapcode 查詢工具

純前端靜態網頁，可直接放上 GitHub Pages（無需後端、無需 build）。

## 檔案
- `index.html` — 主程式（UI + 地圖碼演算法）
- `zonecoords.js` — 日本 1163 個 zone 的邊界座標資料（沿用先前逆向工程成果）

兩個檔案需放在同一層目錄。

## 功能
- **地名搜尋**：可切換 OSM（Nominatim，免費免金鑰）與 Google。OSM 每筆結果附「在 Google 地圖開啟」連結。
- **經緯度**：直接輸入，或貼上「35.6586, 139.7454」自動拆分。
- **點地圖選位置**：啟用點選模式後於地圖按一下即算出地圖碼。
- **反查**：輸入地圖碼定位到網格中心（誤差約 ±15 公尺）。
- 查詢紀錄存於瀏覽器 localStorage。
- iOS liquid glass 介面，設定中可開啟「玻璃折射」進階效果、調整模糊與透明度。
- 電腦與手機皆已適配（窄螢幕地圖移到下方）。

## 演算法說明
地圖碼採三層網格切分：zone → block(30×30) → unit(30×30) → core(10×10)。
編碼／反查邏輯均已驗證，城市座標 round-trip 誤差 < 2 公尺。

## 關於 Google 搜尋
純前端無法安全內嵌 Google API 金鑰，故 Google 模式以「開啟 Google 地圖搜尋頁」方式呈現；
若日後要在程式內直接取得 Google 座標，需自建後端代理金鑰。
